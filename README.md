# 股票失误反省录

记录每一次误操作，让下一次少犯错。

一个极简微信小程序，用来记录股票交易中的误操作，并用亏损比例和影响仓位统计实际损失，帮助股民复盘、减少重复犯错。

## 功能

- 本地记录每次误操作：日期、亏损比例、影响仓位、标的、失误原因、复盘备注。
- 概览页按月历展示每天是否有误操作，并提供近 3 个月和自定义范围统计。
- 统计按失误原因分组，帮助定位最容易造成损失的行为。
- 历史页支持按时间范围查询、编辑和删除记录。
- 备份页支持导出 JSON，以及导入合并或覆盖本地数据。
- 不依赖后台服务器，数据保存在微信小程序本地缓存。

## 使用

1. 打开微信开发者工具。
2. 导入项目目录：`/Users/bingliang/work/proj_TradeRecord`。
3. AppID 可先使用测试号或项目配置里的游客模式。
4. 编译后从“新增”页记录数据，再回到“概览”查看统计。

## 统计口径

- 实际损失 = 亏损比例 × 影响仓位。
- 统计结果以满仓口径展示，例如亏损 10%、影响仓位 30%，实际损失为 -3%。
- 近 3 个月按周统计；自定义范围按跨度自动选择每柱 1 天、1 周或 30 天。
