# TradeRecord

一个极简微信小程序，用来记录股票交易中的误操作和正确操作，并用百分比统计损失、收益和净结果。

## 功能

- 本地记录每笔操作：日期、标的、操作、原因、损失/收益比例、复盘备注。
- 首页展示本月误操作/正确操作简图，以及近 1 月、半年、1 年统计。
- 历史页支持按全部、误操作、正确操作筛选和删除。
- 备份页支持导出 JSON，以及导入合并或覆盖本地数据。
- 不依赖后台服务器，数据保存在微信小程序本地缓存。

## 使用

1. 打开微信开发者工具。
2. 导入项目目录：`/Users/bingliang/work/proj_TradeRecord`。
3. AppID 可先使用测试号或项目配置里的游客模式。
4. 编译后从“记录”页新增数据，再回到“概览”查看统计。

## 统计口径

- 误操作的百分比计入损失。
- 正确操作的百分比计入收益。
- 净结果 = 正确操作收益百分比 - 误操作损失百分比。
- 近 1 月、半年、1 年均按记录日期滚动统计。
